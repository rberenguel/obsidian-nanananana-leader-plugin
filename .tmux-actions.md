# Actions (Obsidian plugin)

## [`c`] Open VS Code

```
code .
```

## [`g`] Open Github

## [`o`] Open folder

```
open .
```

## [`b`] Run build

```
npm run build
```

## [`f`] Run prettier

```
prettier . --write
```

---

## `Version`

```
echo "-v$(jq -r .version manifest.json)"
```

> `./release.sh `
